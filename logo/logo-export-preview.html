Export Instructions for JPG Versions:

  1. Browser Screenshot Method: Right-click each logo container and "Save image as" or use browser dev tools to screenshot the individual logo sections.
  2. SVG to JPG Conversion: Use online tools like convertio.co or cloudconvert.com to convert the SVG files to JPG.
  3. Command Line (if ImageMagick installed):
    convert akili-stacked.svg akili-stacked.jpg
    convert akili-horizontal-compact.svg akili-horizontal-compact.jpg
    convert akili-trademark-only.svg akili-trademark-only.jpg
  4. Design Tools: Open SVG files in Figma, Adobe Illustrator, or Sketch and export as JPG with desired resolution (recommended: 300 DPI for print, 72 DPI for web).

Recommended JPG settings:

  • Quality: 90-95% for best results
  • Background: White or transparent (PNG preferred for transparent)
  • Resolution: 300 DPI for print materials, 72-96 DPI for web use
