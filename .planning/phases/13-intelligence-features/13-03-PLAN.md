- `marriage-relationship-risk`: "Consider prenuptial/postnuptial agreements" / "Establish family wealth education program for spouses"
- `succession-planning`: "Create formal succession timeline" / "Identify and develop next-generation leaders" / "Document transition procedures for key roles"
- `behavior-standards`: "Codify family values and expected behaviors" / "Establish consequences for policy violations"
- `business-involvement`: "Define family employment policies" / "Create compensation guidelines for family members in business"
- `documentation-communication`: "Schedule quarterly family governance meetings" / "Create centralized document repository"

Each recommendation gets a priority: 'high' if the pillar score is critical (<=3.0), 'medium' if moderate (<=5.0), 'low' otherwise.
