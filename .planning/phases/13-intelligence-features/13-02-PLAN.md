Render a grid of 4 stat items in a Card (2 cols on mobile, 4 cols on desktop):
- Total Families (totalFamilies)
- Families at Risk (familiesAtRisk, with destructive text color if > 0)
- Critical Issues (criticalCount, with destructive text color if > 0)
- Total Risk Indicators (portfolioRisksCount)

Use shadcn Card component. Follow the same stat card pattern as `MetricsCards.tsx` in the dashboard.

2. **PortfolioRiskList.tsx** (`"use client"`)
Props: `{ risks: RiskIndicator[] }` (import type from intelligence types)

Render a table-like list of risk indicators, each row showing:
- Family name (as a Link to `/advisor/intelligence/${risk.familyId}` for drill-down)
- Category name (e.g., "Succession Planning", "Decision-Making")
- Score displayed as a number with 1 decimal (e.g., "2.4 / 10")
- Severity badge using shadcn Badge component:
  - critical: `variant="destructive"`
  - moderate: `variant="warning"` (or `variant="outline"` with amber text if warning doesn't exist)
  - low: `variant="secondary"`

If no risks, show empty state: "No significant governance risks identified across your portfolio."

Limit display to top 20 risks (they're already sorted by severity/score from the query). Show count if more exist.

3. **RiskDistributionChart.tsx** (`"use client"`)
Props: `{ risksByCategory: Record<string, number> }`

Use Recharts BarChart (horizontal via `layout="vertical"`) to show risk count per governance category. Import CATEGORY_LABELS from `@/lib/analytics/formatters` for display names. Use the same color-coding convention from CategoryBreakdownChart: bars colored by risk count intensity. Wrap in ChartContainer from `@/components/analytics/ChartContainer`.

If all counts are 0, show "No risks to display" message instead of chart.
