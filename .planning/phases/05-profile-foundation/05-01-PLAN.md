<task type="auto">
  <name>Task 2: Server Actions for household member CRUD</name>
  <files>src/lib/actions/profile-actions.ts</files>
  <action>
Create src/lib/actions/profile-actions.ts with 'use server' directive.

Import: auth from @/lib/auth, prisma from @/lib/db, householdMemberSchema and updateHouseholdMemberSchema from @/lib/schemas/profile, revalidatePath from next/cache.

Helper: async function getAuthUserId() that calls auth(), checks session?.user?.id, throws Error("Not authenticated") if missing, returns userId.

Export async function getHouseholdMembers():
- Get userId via getAuthUserId()
- prisma.householdMember.findMany where userId, orderBy createdAt asc
- Return { success: true, members }
- Catch: return { success: false, error: message }

Export async function createHouseholdMember(data: unknown):
- Get userId
- Validate with householdMemberSchema.safeParse(data)
- If invalid, return { success: false, errors: validatedFields.error.flatten().fieldErrors }
- prisma.householdMember.create with { ...validatedFields.data, userId }
- revalidatePath('/profiles')
- Return { success: true, member }
- Catch: return { success: false, error: message }

Export async function updateHouseholdMember(id: string, data: unknown):
- Get userId
- Validate with updateHouseholdMemberSchema.safeParse(data)
- If invalid, return errors
- prisma.householdMember.update where { id, userId } (ensure ownership), data: validatedFields.data
- revalidatePath('/profiles')
- Return { success: true, member }
- Catch: return { success: false, error: message }

Export async function deleteHouseholdMember(id: string):
- Get userId
- prisma.householdMember.delete where { id, userId } (ensure ownership)
- revalidatePath('/profiles')
- Return { success: true }
- Catch: return { success: false, error: message }

Key: All operations enforce userId ownership so users can only manage their own household members. Use compound where clauses { id, userId } on update/delete.
  </action>
