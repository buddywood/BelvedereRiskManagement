**GovernanceRecommendations.tsx:**
- Accept props: `members: Array<{ fullName: string; governanceRoles: string[]; relationship: string }>` and `missingControls: Array<{ category: string; recommendation: string; severity: string }>`
- Render a Page with header "Family Governance Recommendations"
- Group recommendations by governance role:
  - For each role present in household (DECISION_MAKER, SUCCESSOR, TRUSTEE, etc.), show a subsection with the member names who hold that role and role-specific recommendations

  - SUCCESSOR: Recommendations about succession planning, mentorship, capability development
  - TRUSTEE: Recommendations about fiduciary responsibilities, trust governance
  - ADVISOR: Recommendations about advisory board structure, meeting cadence
  - BENEFICIARY: Recommendations about distribution policies, beneficiary rights
  - EXECUTOR: Recommendations about estate execution, document management
- Cross-reference with missingControls: if a missing control's category relates to a role's domain, include it under that role's section
- If no members have governance roles, show a generic recommendation to assign governance roles
